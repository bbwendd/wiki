# 論壇研討會籌備流程

## 目的

協助籌備論壇、研討會、成果發表會，包含講者邀請、議程、宣傳、場務、現場執行與結案。

## 輸入

- 活動名稱、日期、場地、主題、目標對象。
- 講者名單、國外講者需求、合作單位。
- 預算、宣傳素材、報名資料。

## 處理步驟

1. 確認活動定位與成功標準。
2. 建立議程與講者追蹤表。
3. 產生中文或英文邀請草稿。
4. 規劃宣傳、報名、場務、接待與設備。
5. 建立總時程、任務分工與 run down。
6. 會後整理成果、感謝信、照片錄影、結案資料。

## 輸出

- 論壇籌備清單。
- 講者追蹤表。
- 英文邀請信。
- 議程草案。
- 任務分工表。
- Run down 與結案清單。

## 需要確認的地方

- 講者是否已確認。
- 國外講者交通、住宿、reimbursement。
- 活動預算、場地、宣傳時程。

## 常見失敗情境

- 只有議程，沒有場務與結案。
- 講者承諾未追蹤來源。
- 英文邀請信未確認講者角色與時間。
