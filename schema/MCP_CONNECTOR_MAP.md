# MCP Connector 能力對照表

本檔定義第二大腦可使用的外部工具與連接器邊界。所有對外行動仍維持「先草稿後確認」。

## 核心原則

- Connector 用來讀取、整理、建立草稿或準備操作，不代表可以自動對外送出。
- 寄信、傳訊息、修改行程、刪除檔案、公開發布、提交承諾前，必須再次取得使用者明確確認。
- 無 connector 或授權不足時，改用使用者貼上的文字、截圖、附件或手動匯出資料。
- 無法驗證的資料必須標示為 `待確認`。

## Connector 對照

| 工具 / Connector | 可用於 | 可產出 | 禁止 / 需確認 |
|---|---|---|---|
| Gmail | Email 瀏覽、搜尋、摘要、待回覆判斷、回覆草稿 | 待回覆清單、Email 草稿、追蹤事項 | 不自動寄出、不自動封存或刪除 |
| Google Calendar | 行程查詢、空檔比對、會議準備、排程草稿 | 可行時段、會議摘要、排程建議 | 不自動建立、修改或取消會議 |
| Google Drive / Docs | 查找檔案、讀取文件、建立或整理文件草稿 | 文件草稿、資料摘要、來源清單 | 不刪除、不移動重要檔案、不分享權限，除非另行確認 |
| Google Sheets / Excel | 表格建立、欄位設計、公式規劃、資料整理 | 追蹤表、預算表、名單、規格表 | 不覆蓋重要資料，除非另行確認 |
| Google Slides / Presentations | 簡報大綱、投影片草稿、簡報檔製作 | 簡報架構、投影片內容、素材需求 | 不代表已完成正式視覺審稿，需另行檢查 |
| GitHub | repo 檢查、issue / PR、提交、推送、版本紀錄 | commit、push、PR 草稿、變更摘要 | destructive git 操作需明確確認 |
| Canva | DM、社群圖、簡報視覺稿、設計改版 | 可編輯設計草稿、尺寸變體 | 發布、分享或品牌承諾需確認 |
| Slack | 訊息搜尋、摘要、草稿、頻道整理 | Slack 回覆草稿、摘要、待辦 | 不自動發送或承諾 |
| Line | 目前若無 connector，以貼上文字、截圖或轉貼內容處理 | Line 回覆草稿、語氣版本 | 不自動傳送 |
| Browser / Web Search | 專業資料搜尋、政府標案資料、背景查證 | 來源摘要、引用、風險提醒 | 高變動或高風險資料需保留來源與日期 |
| 音檔轉錄工具 | 音檔轉逐字稿、會議紀錄整理 | 逐字稿、摘要、決議、待辦 | 轉錄品質不足需標示，不能假裝逐字精準 |

## 常見任務路由

| 任務 | 優先 Connector | 備援方式 | 必要輸出 |
|---|---|---|---|
| 瀏覽 Email 並回覆 | Gmail | 使用者貼上 email 內容 | 回覆草稿與是否需回覆判斷 |
| 行程與會議排程 | Google Calendar | 使用者提供可用時段 | 候選時段與確認點 |
| 做簡報 | Google Slides / Presentations / Canva | Markdown 大綱 | 頁面架構、素材需求、簡報草稿 |
| 做 Excel 表格 | Google Sheets / Spreadsheets | CSV / Markdown 表格 | 欄位、公式、篩選與使用說明 |
| 政府標案閱讀 | Browser / PDF / Drive | 使用者提供 PDF 或網址 | 摘要、資格、文件、時程、風險 |
| 論壇研討會籌備 | Calendar / Gmail / Drive / Sheets / Canva | 手動資料夾與表格 | 講者、議程、分工、時程、追蹤表 |

```md
## Connector 狀態

- 嘗試使用：
- 結果：成功 / 失敗 / 授權不足 / 無 connector
- 未驗證資料：
- 替代處理：
- 需要使用者提供：
```
