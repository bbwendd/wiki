# 活動籌備規則

本檔用於論壇、研討會、成果發表會與國外講者邀請。

## 活動籌備主軸

1. 活動定位：目的、對象、主題、成果。
2. 議程設計：時間、場次、講者、主持人、交流安排。
3. 講者管理：邀請、確認、簡介、題目、簡報、交通住宿。
4. 報名與宣傳：DM、報名頁、社群、Email、媒體。
5. 行政與場務：場地、設備、餐飲、接待、翻譯、動線。
6. 現場執行：run down、工作人員、突發狀況。
7. 結案：成果報告、照片、錄影、新聞稿、感謝信。

## 國外講者溝通

- 英文信件保持正式、清楚、簡短。
- 每次往來都記錄：日期、主旨、對方回覆、承諾、待確認。
- 必須追蹤：時區、航班、住宿、接送、講題、簡報、肖像授權、酬勞或 reimbursement。

## 論壇 / 研討會任務分類

| 類別 | 任務 |
|---|---|
| 議程 | 主題、場次、時間、主持人、講者 |
| 講者 | 邀請、確認、簡介、講題、簡報、交通住宿 |
| 宣傳 | DM、報名頁、新聞稿、社群、邀請名單 |
| 行政 | 場地、設備、餐飲、翻譯、採購 |
| 現場 | run down、接待、報到、攝影、控場 |
| 結案 | 成果報告、感謝信、核銷、資料歸檔 |

## 完成標準

- 有總時程。
- 有講者追蹤表。
- 有任務分工表。
- 有當天 run down。
- 有會後成果與結案清單。
