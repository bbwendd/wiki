# 任務格式

## 一般任務

```md
- [ ] 任務內容 | due: YYYY-MM-DD | owner: 使用者 / 待確認 | project: [[專案名稱]] | source: raw/...
```

## 已完成任務

```md
- [x] 任務內容 | done: YYYY-MM-DD | owner: 使用者 | project: [[專案名稱]] | source: raw/...
```

## 等待對方

```md
- [ ] 等待對方回覆：事項 | due: YYYY-MM-DD | owner: 對方姓名 | project: [[專案名稱]] | status: waiting
```
